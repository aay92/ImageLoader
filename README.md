Добрый день!

Приложения для загрузки картинок из сети!

Какие возможности есть: 

 - Автоматическая пагинация, 
 - Обновление контента по свайпу сверху вниз,
 - Возможность сохранить/удалить картинки из вкладки "Любимые".
